# 新闻看板 - 前端需求文档

## 1. 概述

为新闻采集项目开发前端看板页面，展示最新批次中高评分的新闻，帮助操盘手和散户快速捕捉市场热点。

## 2. 目标用户

- 操盘手：快速识别高影响力新闻，辅助交易决策
- 散户：了解市场热点，跟随资金方向

## 3. 数据来源

- 数据表：`importance`
- 筛选条件：
  - 最新批次（按 `created_at` 分组，取最新批次）
  - 按 `importance_score` 排序，取前 10 条
- 展示字段：
  - 标题 `title`
  - 关联板块 `related_sectors`
  - 评分 `importance_score`
  - 发布时板块指数 `publish_sector_values`
  - 当前板块指数 `current_sector_values`
  - 指数变化 `publish_sector_values` vs `current_sector_values` 计算差值
  - 摘要 `summary`

## 4. UI 设计

### 4.1 技术栈

- **框架**：Next.js + React
- **样式**：Tailwind CSS
- **组件库**：shadcn/ui
- **数据获取**：直接读取 SQLite 数据库

### 4.2 视觉风格

- **主题**：深色主题（适合长时间盯盘）
- **背景色**：深灰/黑色（#0a0a0a 或类似）
- **强调色**：高饱和度颜色突出重要信息
  - 涨（红） / 跌（绿）- 符合 A 股颜色习惯
- **卡片式布局**：每条新闻一张卡片，信息层次分明

### 4.3 页面布局

```

│  Header: 新闻看板标题 + 更新时间                      │
├─────────────────────────────────────────────────────┤
│  筛选/排序区域（可选，暂时不做）                        │
├─────────────────────────────────────────────────────┤
│  ┌──────────┐ ┌──────────┐ ┌──────────┐            │
│  │ 新闻卡片1 │ │ 新闻卡片2 │ │ 新闻卡片3 │  ...       │
│  │          │ │          │ │          │            │
│  │ 标题     │ │ 标题     │ │ 标题     │            │
│  │ 板块     │ │ 板块     │ │ 板块     │            │

│  │ 指数变化 │ │ 指数变化 │ │ 指数变化 │            │

```

### 4.4 卡片内容

每张卡片包含：

| 元素 | 说明 |
|------|------|
| 标题 | 新闻标题，超长截断 |
| 评分徽章 | 数字 1-10，带颜色（高分更醒目） |
| 关联板块 | 板块名标签，点击可过滤 |
| 指数变化 | publish vs current 差值，红涨绿跌 |
| 摘要 | 一句话概括新闻影响 |
| 跳转链接 | 点击跳转到原文 |

## 5. 功能

### 5.1 核心功能

1. **展示最新前 10 新闻**
   - 自动获取最新批次
   - 按评分排序

2. **指数变化计算**
   - 从 `publish_sector_values` 和 `current_sector_values` 解析
   - 计算涨跌百分比

3. **刷新机制**
   - 页面手动刷新
   - 可选：自动刷新间隔（暂不做）

### 5.2 交互

- 点击卡片跳转到原文
- 指数变化用颜色区分涨跌
