# 新闻看板 - Flutter App 规格文档

## 1. 项目概述

- **项目名称**：新闻看板 (NewsBoard)
- **类型**：移动端应用（Android APK）
- **核心功能**：展示最新批次中高评分的新闻，帮助操盘手和散户快速捕捉市场热点
- **目标用户**：操盘手、散户

## 2. 技术栈

| 项目 | 技术 |
|------|------|
| 框架 | Flutter 3.x |
| 语言 | Dart |
| 状态管理 | Riverpod / Provider |
| HTTP | Dio |
| 本地存储 | SQLite (sqflite) |
| UI 设计 | Material Design 3 |
| 架构 | Clean Architecture |

## 3. UI/UX 设计

### 3.1 视觉风格

- **主题**：深色主题 + 玻璃拟态（Glassmorphism）
- **配色**：
  - 背景：深黑渐变 (#0D0D0D → #1A1A1A)
  - 强调色：鲜艳的红/绿（A股风格）
  - 文字：白色主文字 + 灰色辅助文字
- **圆角**：大圆角（16-24px）营造柔和感
- **阴影**：柔和阴影 + 毛玻璃效果

### 3.2 布局

### 3.3 卡片设计

- **玻璃拟态卡片**：
  - 半透明背景（20% 白）
  - 模糊效果
  - 细边框（1% 白）
  - 阴影

- **动效**：
  - 卡片入场动画（交错滑入）
  - 指数变化数字动画
  - 点击涟漪效果
  - 下拉刷新动画

## 4. 功能

### 4.1 核心功能

1. **新闻列表**
   - 获取最新批次评分前 10 条
   - 按评分排序

2. **指数变化展示**
   - publish vs current 差值
   - 动画数字
   - 颜色区分涨跌

3. **下拉刷新**
   - 手动刷新数据

4. **跳转原文**
   - 点击卡片打开新闻链接

### 4.2 数据展示

| 字段 | 展示方式 |
|------|----------|
| title | 大字标题，渐变色 |
| importance_score | 圆形徽章，颜色区分 |
| related_sectors | 标签列表 |
| publish_sector_values | 板块名 + 指数值 |
| current_sector_values | 当前指数 + 变化 |
| summary | 灰色辅助文字 |

## 5. 页面

### 5.1 首页（新闻列表）

- 顶部 AppBar（标题 + 刷新）
- 批次信息卡片
- 新闻卡片列表
- 下拉刷新
- 上拉加载更多（可选）

### 5.2 新闻详情页

- 点击卡片进入
- 完整摘要
- 板块详细数据
- 跳转原文按钮
