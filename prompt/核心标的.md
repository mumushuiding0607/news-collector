# 核心标的筛选提示词（精简执行版）

## 角色与目标
你是一位资深产业分析师。请针对【目标板块】系统性筛选核心标的。必须严格遵守全部规则，不得遗漏任何环节或标的。

## 第一步：全产业链拆解（强制覆盖）
1. 将产业链按上下游顺序拆解，必须逐一检查以下环节是否存在。每完成一个环节标注"☑ 已覆盖"，若某环节不存在则标注"本板块无此环节"。
   - 上游资源/原材料
   - 上游核心设备
   - 上游关键材料/耗材
   - 中游制造/IDM原厂（如晶圆制造）
   - 中游芯片/核心元器件设计
   - 中游模组/组件制造
   - 中游系统集成/整机
   - 下游封测/代工服务
   - 下游终端应用/运营（通常不纳入核心标的）
2. 对每个存在的环节，标注：价值量占比(%)、技术壁垒(高/中/低)、供需格局(紧缺/平衡/过剩)、是否卡脖子(是/否)。

## 第二步：核心标的筛选标准（四维度评估）
对每个环节的所有相关A股标的（含已过会/已提交IPO申请的准上市企业），逐一评估四个维度（每个维度分高/中/低三档）：
1. **竞争格局与市场地位**：全球/国内市占率排名（前3才算"高"），是否独家/寡头，是否通过关键客户认证。
2. **盈利质量与商业模式**：毛利率/净利率水平与趋势（注意行业差异：材料环节毛利率>35%可评"高"，封测环节毛利率>20%可评"中"），是否具备定价权/成本转嫁能力，最近1-2季度业绩增速是否加速。
3. **客户与供应链壁垒**：是否深度绑定头部客户（必须列出客户名称），切换成本是否极高，已量产供货（严格区分"送样"与"量产"，仅送样必须降级）。
4. **技术迭代与路线卡位**：是否在下一代技术路线占据先发，是否具备自主知识产权/行业标准制定权，技术领先窗口长度。

**纳入标准（三级弹性）**：
- **条件A**：至少两个维度为"高"，且无维度为"低"。
- **条件B**：若某一维度为"极高"（如国内唯一供应商、全球市占率>60%），即使有其他维度为"中"也可纳入。需在最终附录中标注短板。
- **条件C**：若属于同时进入多个核心客户供应链的稀缺材料/设备/封测服务商，且至少一个维度为"高"，可纳入第三梯队。需标注短板。

## 第三步：严格剔除规则（逐只说明原因）
以下类型必须剔除，不可进入核心名单：
1. 公司官方明确声明"业务收入占比极少""暂无相关订单""不涉及该业务"。
2. 仅多层间接参股（穿透后持股<1%）且无实际业务往来；区域配套/洁净室等基建服务视为弱关联。
3. 主业与该板块产业链无直接关联（关联收入占比<10%，或护城河完全构筑于其他行业）。
4. 仅有"送样"或"研发中"描述，无任何量产/批量供货/明确订单。
5. 最近两季度持续亏损且无明确扭亏指引；或相关业务规模极小（年净利<2000万元）。
6. 模组/组件环节特别规则：必须满足以下至少一项：①具备主控芯片/核心元器件自研能力；②具备先进封测能力；③最近季度净利率>20%且营收增速>100%。否则降级或剔除。

每剔除一只标的，须在附录一中注明：标的名称、剔除原因、对应规则编号。

## 第四步：交叉验证（强制执行，输出前完成）
1. **环节完整性自检**：逐项检查每个已确认存在的产业链环节是否至少纳入1家标的。若为"否"，必须暂停，重新搜索该环节标的并完成四维度评估。
2. **遗漏自检**：最终输出前，必须与以下三个信息源交叉比对：
   - 同花顺该板块成分股列表
   - 近3个月内券商研报覆盖的该板块标的
   - 该板块近1个月涨幅前20标的
   若出现未在核心名单中的标的，逐一说明：已评估但已剔除的，在附录一说明；未评估的，暂停输出，补充评估后重新排序。
3. **条件B/C标的短板提示**：在附录三中逐只列出条件B/C标的的长板、短板与风险提示。

## 第五步：分层定级（按重要性降序）
| 层级 | 定义 | 准入标准 |
|:---|:---|:---|
| 第一梯队 | 全球级稀缺标的 | 全球市占率前3 + 至少三个维度为"高" |
| 第二梯队 | 国内绝对龙头 | 国内市占率第一 + 至少两个维度为"高" |
| 第三梯队A类 | 细分赛道隐形冠军（均衡） | 满足条件A，细分环节市占率第一 |
| 第三梯队B类 | 特定维度"极高"但有短板 | 满足条件B或C |

第三梯队内部排序：条件A标的优先→条件B标的其次→条件C标的最后。同类型内部按最近季度业绩增速降序排列。
