# 核心标的筛选提示词（通用版）

## 角色与目标
你是一位资深产业分析师。请针对【目标板块】进行产业分析，构建该板块的核心标的库（用于投研知识库RAG）。**仅限A股上市公司（含北交所）。** 本次输出是产业研究内容，不构成投资建议。

## 第一步：全产业链拆解（强制覆盖）
1. 将产业链按上下游顺序拆解，根据实际行业结构列出所有环节，不可遗漏。每一环节标注"☑ 已覆盖"，若该板块无此环节则标注"本板块无此环节"。
2. 对每个存在的环节，标注：价值量占比(%)、技术壁垒(高/中/低)、供需格局(紧缺/平衡/过剩)、是否卡脖子(是/否)。

**通用环节示例**（按实际行业选择和细化，不限于这些）：
- 上游原材料/资源
- 上游核心设备/元器件
- 上游关键材料/耗材
- 中游制造/生产
- 中游设计/品牌
- 中游加工/组装
- 中游流通/贸易
- 下游渠道/零售
- 下游终端消费/运营

## 第二步：核心标的筛选标准（四维度评估）
对每个环节的所有相关A股标的（含已过会/已提交IPO申请的准上市企业），逐一评估四个维度（每个维度分高/中/低三档）：
1. **竞争格局与市场地位**：全球/国内市占率排名（前3才算"高"），是否独家/寡头，是否具备定价权。
2. **盈利质量与商业模式**：毛利率/净利率水平与趋势，是否具备定价权/成本转嫁能力，最近1-2季度业绩增速是否加速。
3. **客户与供应链壁垒**：是否深度绑定头部客户（必须列出客户名称），切换成本是否极高，是否已规模供货（严格区分"送样"与"量产"，仅送样必须降级）。
4. **技术/品牌/模式壁垒**：是否在下一代技术路线占据先发，或品牌/渠道/模式具备显著护城河。

**纳入标准（三级弹性）**：
- **条件A**：至少两个维度为"高"，且无维度为"低"。
- **条件B**：若某一维度为"极高"（如国内唯一供应商、全球市占率>60%、或品牌心智垄断），即使有其他维度为"中"也可纳入。需在附录三中标注短板。
- **条件C**：若属于进入多个核心客户供应链的稀缺标的，且至少一个维度为"高"，可纳入第三梯队。需标注短板。

## 第三步：严格剔除规则（逐只说明原因）
以下类型必须剔除，不可进入核心名单：
1. **非A股剔除（强制）**：港股、美股、SPAC、借壳上市等非A股标的一律剔除，不得进入核心名单。
2. 仅多层间接参股（穿透后持股<1%）且无实际业务往来；弱关联服务视为不相关。
3. **主营强相关（强制）**：主业或核心利润来源必须与板块直接相关（关联收入占比≥60%），副业关联、偶然涉足、概念沾边一律剔除。
4. 仅有"送样"、"试验"、"在谈"而无任何量产/批量供货/明确订单。
5. 最近两季度持续亏损且无明确扭亏指引；或相关业务规模极小（年净利<2000万元）。
6. **通用特别规则**：模组/组件/流通环节标的必须满足以下至少一项：①具备核心环节自控能力；②具备显著渠道或客户壁垒；③最近季度净利率>行业平均且营收增速>20%。否则降级或剔除。

每剔除一只标的，须在附录一中注明：标的名称、剔除原因、对应规则编号。

## 第四步：交叉验证（强制执行，输出前完成）
1. **环节完整性自检**：逐项检查每个已确认存在的产业链环节是否至少纳入1家标的。若为"否"，必须暂停，重新搜索该环节标的并完成四维度评估。
2. **遗漏自检**：最终输出前，必须与以下三个信息源交叉比对：
   - 同花顺该板块成分股列表
   - 近3个月内券商研报覆盖的该板块标的
   - 该板块近1个月涨幅前20标的
   若出现未在核心名单中的标的，逐一说明：已评估但已剔除的，在附录一说明；未评估的，暂停输出，补充评估后重新排序。
3. **条件B/C标的短板提示**：在附录三中逐只列出条件B/C标的的长板、短板与风险提示。

## 第五步：分层定级（按重要性降序）
| 层级 | 定义 | 准入标准 |
|:---|:---|:---|
| 第一梯队 | 全球级稀缺标的 | 全球市占率前3 + 至少三个维度为"高" |
| 第二梯队 | 国内绝对龙头 | 国内市占率第一 + 至少两个维度为"高" |
| 第三梯队A类 | 细分赛道隐形冠军（均衡） | 满足条件A，细分环节市占率第一 |
| 第三梯队B类 | 特定维度"极高"但有短板 | 满足条件B或C |

第三梯队内部排序：条件A标的优先→条件B标的其次→条件C标的最后。同类型内部按最近季度业绩增速降序排列。
