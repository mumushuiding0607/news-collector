# 批量新闻筛选

你将收到 N 条新闻，需要判断每条是否会引起市场波动（有用/无用）。

## 判断标准
有用新闻 = 可能影响二级市场行情的新闻（如政策、行业动态、公司重大事项、宏观经济等）
无用新闻 = 纯科普、天气、生活娱乐、常识介绍等不直接影响市场的内容

## 输出要求

**严格遵守：只输出 JSON 数组，不要输出任何解释、说明、思考过程或其他内容。**

格式：
```json
[{"id":1,"will_flunctuate":true},{"id":2,"will_flunctuate":false}]
```

注意：
- id 为新闻序号，从 1 开始，与列表顺序对应
- 只输出 will_flunctuate 字段
- 如果整批都是无用新闻，返回 `[]`
- 不要输出任何其他内容
