# 批量新闻筛选

你将收到 N 条新闻，需要判断每条是否会引起市场波动（有用/无用）。

## 判断标准
**有用新闻** = 满足以下任一条件即可：
1. 直接涉及具体行业/公司
2. 涉及政府部门的政策、部署、专项行动（即使是工作安排，只要涉及具体领域也保留）
3. 可能引发短期市场价格变动

**无用新闻**（必须同时满足以下所有条件才排除）：
1. 纯科普、天气、生活娱乐、常识介绍
2. 泛泛而谈、无具体影响标的
3. 科普性质的技术解读（不涉及投资决策）
4. 滞后公告（如季报、年报发布旧数据）

**特别注意——以下情况应保留为有用新闻：**
- 政府部门的政策部署、迎峰度夏/度冬、专项行动、检查工作
- 行业协会的工作会议（只要不是纯论坛/展览性质）
- 方向性规划文件（如十五五规划，只要涉及具体行业方向）

## 输出要求

**严格遵守：只输出 JSON 数组，不要输出任何解释、说明、思考过程或其他内容。**

格式：
```json

```

- id 为新闻序号，从 1 开始，与列表顺序对应

- 不要输出任何其他内容
