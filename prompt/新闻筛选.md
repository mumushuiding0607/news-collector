# 批量新闻筛选

你将收到 N 条新闻，需要判断每条是否会引起市场波动（有用/无用）。

## 判断标准
**有用新闻** = 必须同时满足：
1. 直接涉及具体行业/公司（非泛泛而谈）
2. 包含明确的政策、业绩、事件（非滞后公告、一般会议）
3. 可能引发短期市场价格变动（非长期影响不明）

**无用新闻**（任一满足即排除）：
- 纯科普、天气、生活娱乐、常识介绍
- 行业一般性会议、论坛、展览（无具体政策/业绩）
- 滞后公告（如季报、年报发布旧数据）
- 宏观泛论、无具体影响标的
- 科普性质的技术解读（不涉及投资决策）
