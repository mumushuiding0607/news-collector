# 指令：事件评估与板块识别

## 前置判断

先判断新闻属于哪一类，否则不评分：

**新建事件**：首次宣布/发布/披露，时间是今天或近期，市场未定价 → 正常评分
**二手评论**：引用旧事件（>1个月前）并发表看法，如大佬表态、行业分析 → 降档：intensity ≤3，duration ≤ 中期，必要时 will_flunctuate=false

判断线索：时间标志词（几个月前、半年前、近年来、此前、已经）、引用"某事件"作为背景来发表评论

## 评分维度

1. **影响方向**：积极 / 消极
2. **影响强度**：1～5 级（5=改变行业格局的重大事件）
3. **预期波幅**：±%（取绝对值，换算：±2%=1，±5%=2，±10%=3，±20%=4，>±20%=5）
4. **持续时间**：超短期/短期/中期/长期/永久（对应1-5分）

评分 = 影响强度级数 × 0.7 + 波幅得分 × 0.15 + 持续时间得分 × 0.15

## 四维定性

- 本质属性：宏大叙事 或 消息
- 参与者结构：全市场资金 或 单一资金
- 清晰度：模糊长逻辑 或 清晰短期可兑现
- 基本面影响深度：改变估值模型/增长曲线 或 仅影响增长曲线斜率

定性结论：重大预期/中等预期/较小预期；趋势主升/脉冲式/其他

## 板块识别

输出1-2个最受益板块，不能超过2个，**必须匹配同花顺标准板块名**。

  "importance_score": 1-10,
  "reason": "30-80字评分依据",
  "direction": "积极或消极",
  "intensity": 1-5,
  "expected_change": "±X%",
  "duration": "超短期/短期/中期/长期/永久",
