# 指令：事件评估与板块识别

对每个事件按顺序执行以下步骤，最后按指定格式输出。

## 1. 基础评估
- 影响方向：积极 / 消极
- 影响强度：1～5 级
- 预期价格波幅：±%（取绝对值）
- 持续时间：超短期 / 短期 / 中期 / 长期 / 永久

## 2. 四维预期定性
- 本质属性：宏大叙事 或 消息
- 参与者结构：全市场资金 或 单一资金
- 清晰度：模糊长逻辑 或 清晰短期可兑现
- 基本面影响深度：改变估值模型/增长曲线 或 仅影响增长曲线斜率

## 3. 定性结论
- 预期级别：重大预期 / 中等预期 / 较小预期
- 逻辑依据：用一句话概括核心理由
- 行情模式：趋势主升 / 脉冲式 / 其他（具体说明）

## 4. 综合评分计算
评分 = 影响强度级数 × 0.7 + 波幅得分 × 0.15 + 持续时间得分 × 0.15

得分换算规则：
- 影响强度：直接使用 1～5 的数值
- 波幅得分：0～2% = 1，2～5% = 2，5～10% = 3，10～20% = 4，>20% = 5
- 持续时间得分：超短期=1，短期=2，中期=3，长期=4，永久=5

## 5. 最受益板块
从事件直接映射的核心行业中，确定唯一一个最受益板块名称。
